# NFT Galaxy - Collection Viewer

Explore and discover top NFT collections with floor prices, volume, and rankings

## Features

- Top 20 NFT collections in card grid
- Floor price and 24h volume display
- Collection rankings with emoji icons
- Chain platform badges
- Hover animations with elevation effect
- Violet/cyan gradient dark theme

## Tech Stack

- **Framework:** Next.js 16 (App Router, TypeScript)
- **UI:** shadcn/ui + Tailwind CSS
- **Animations:** Framer Motion
- **API:** CoinGecko NFT API (Free)
- **Deployment:** Vercel

🔗 **Live Demo:** [https://nft-collection-viewer-omega.vercel.app](https://nft-collection-viewer-omega.vercel.app)
